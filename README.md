# Voice-Control

macOS voice assistant for ephemeral Gemini chats — push-to-talk, red-hat activation, day-to-day work helper.

**Status:** Spec phase. See [docs/SPEC.md](docs/SPEC.md).

## Overview

- Native macOS menu bar + floating panel (not a Linux container foundation)
- Temporary sessions with Gemini (no persistence by default)
- Apple Speech for STT, streaming Gemini API, spoken/text replies
